Auth API Documentation

Welcome to the Auth API documentation. This API provides endpoints for managing authentication in your e-commerce application.

Base URL

http://localhost:3000/api/auth

Authentication

Some endpoints require authentication. Admin endpoints require admin privileges.

Authorization: Bearer <your_token>

Response Format

All responses follow a standard format:

{
  "success": true,
  "data": { ... } // Response data
}

// Error response
{
  "success": false,
  "error": "Error message"
}
