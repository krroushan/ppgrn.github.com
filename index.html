PhonePe Payment Gateway Integration

PhonePe Payment Gateway Integration Guide for React Native

This guide will help you integrate PhonePe payment gateway in your React Native application.

1Installation

npm install react-native-phonepe-pg

2Configuration

Create a configuration file (api/server.js):

// PhonePe Configuration
const ENV = 'UAT';  // or 'PROD' for production
const BASE_URL = 'https://api-preprod.phonepe.com/apis/pg-sandbox';
const CLIENT_ID = 'YOUR_CLIENT_ID';
const CLIENT_SECRET = 'YOUR_CLIENT_SECRET';
const CLIENT_VERSION = 1;
const MERCHANT_ID = 'YOUR_MERCHANT_ID';

// Token cache
let tokenCache = {
    access_token: null,
    expires_at: 0
};

3Auth Token Generation

export const generateAuthToken = async () => {
    try {
        const formData = new URLSearchParams();
        formData.append('client_id', CLIENT_ID);
        formData.append('client_secret', CLIENT_SECRET);
        formData.append('client_version', CLIENT_VERSION.toString());
        formData.append('grant_type', 'client_credentials');

        const response = await fetch(`${BASE_URL}/v1/oauth/token`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/x-www-form-urlencoded',
                'Accept': 'application/json'
            },
            body: formData.toString()
        });

        const data = await response.json();
        if (data.access_token) {
            tokenCache = {
                access_token: data.access_token,
                expires_at: data.expires_at
            };
            return data.access_token;
        }
        throw new Error('No access token in response');
    } catch (error) {
        throw error;
    }
};

4Order Creation

export const createOrder = async (amount) => {
    try {
        const token = tokenCache.access_token;
        if (!token) {
            throw new Error('No auth token available');
        }

        const merchantOrderId = `ORDER${Date.now()}${Math.random().toString(36).slice(2, 6)}`;

        const orderPayload = {
            merchantOrderId: merchantOrderId,
            amount: amount,
            expireAfter: 1200,
            metaInfo: {
                udf1: "React Native App",
                udf2: merchantOrderId
            },
            paymentFlow: {
                type: "PG_CHECKOUT"
            }
        };

        const response = await fetch(`${BASE_URL}/checkout/v2/sdk/order`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': `O-Bearer ${token}`,
            },
            body: JSON.stringify(orderPayload)
        });

        const data = await response.json();
        return {
            orderId: data.orderId,
            token: data.token,
            merchantOrderId: merchantOrderId,
            state: data.state
        };
    } catch (error) {
        throw error;
    }
};

6Payment Flow Implementation

const startPayment = async () => {
    if (!isSDKInitialized) {
        Alert.alert("Error", "Payment system not initialized");
        return;
    }

    setIsLoading(true);
    try {
        const orderData = await createOrder(100); // Amount in paise
        
        const requestBody = {
            orderId: orderData.orderId,
            merchantId: "YOUR_MERCHANT_ID",
            merchantOrderId: orderData.merchantOrderId,
            token: orderData.token,
            amount: 100,
            paymentMode: {
                type: "PAY_PAGE"
            },
            redirectUrl: "YOUR_REDIRECT_URL",
            callbackUrl: "YOUR_CALLBACK_URL",
            deviceContext: {
                deviceOS: "ANDROID"
            }
        };

        const response = await PhonePePaymentSDK.startTransaction(
            JSON.stringify(requestBody),
            null
        );
        
        handlePaymentStatus(response);
    } catch (error) {
        Alert.alert("Error", "Payment failed. Please try again.");
    } finally {
        setIsLoading(false);
    }
};

API Reference

1. Generate Auth Token

POST Endpoint: /v1/oauth/token

Base URL: https://api-preprod.phonepe.com/apis/pg-sandbox (UAT)

Request Headers:
{
    'Content-Type': 'application/x-www-form-urlencoded',
    'Accept': 'application/json'
}
Request Body:
{
    client_id: 'YOUR_CLIENT_ID',
    client_secret: 'YOUR_CLIENT_SECRET',
    client_version: '1',
    grant_type: 'client_credentials'
}
Success Response:
{
    "access_token": "eyJhbGciOiJIUzI1...",
    "token_type": "Bearer",
    "expires_in": 3600,
    "expires_at": 1679831543000
}
Error Response:
{
    "error": "invalid_client",
    "error_description": "Invalid client credentials"
}

2. Create Order

POST Endpoint: /checkout/v2/sdk/order

Request Headers:
{

    'Authorization': 'O-Bearer YOUR_ACCESS_TOKEN'

Request Body:
{

    "metaInfo": {

Important Notes:

  • All amounts are in paise (1 INR = 100 paise)
  • Auth tokens typically expire after 1 hour
  • Order expiry can be set between 5 minutes to 24 hours
  • Always implement proper error handling for all API calls
  • Store sensitive credentials securely
  • Implement webhook handling for payment status updates
