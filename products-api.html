Product API Documentation

Welcome to the Product API documentation. This API provides endpoints for managing products in your e-commerce application.

Base URL

http://localhost:3000/api

Authentication

Most endpoints require authentication. Admin endpoints require admin privileges.

Authorization: Bearer <your_token>

Response Format

All responses follow a standard format:

{
  "success": true,
  "data": { ... } // Response data
}

// Error response
{
  "success": false,
  "error": "Error message"
}

Public Endpoints

GET
/api/products
List all products

Returns a paginated list of products with optional filtering.

Query Parameters
Parameter Type Description
page number Page number (default: 1)
limit number Items per page (default: 12)
category string Filter by category ID
primaryCategory string Filter by primary category ID
featured boolean Filter featured products
onSale boolean Filter products on sale
keyword string Search by keyword
seasonal boolean Filter seasonal products
stockStatus string Filter by stock status
minPrice number Minimum price filter
maxPrice number Maximum price filter
inStock boolean Filter in-stock products
sort string Sort order (price-asc, price-desc, name-asc, name-desc, newest, oldest)

GET /api/products
